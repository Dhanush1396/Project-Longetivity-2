# 30-Day Longevity Study Curriculum

## Design Principles
- Each day = one tightly scoped topic
- Progression: evolutionary → molecular → interventional → clinical → frontier
- Each session: ~3,500–5,000 words, inline links throughout, 8–12 cited sources minimum
- No generic overviews — specific studies, specific numbers, specific debates
